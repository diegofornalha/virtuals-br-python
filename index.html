Apresentando o GAME

GAME é um framework modular que permite que agentes planejem ações e tomem decisões de forma autônoma. Um motor de tomada de decisão construído sobre modelos de fundação para potencializar agentes em diferentes ambientes e plataformas.

📦 Instalar SDK 📚 Documentação

Recursos Principais

🌍

Aberto para Todos

Qualquer pessoa pode usar o GAME, independentemente da plataforma ou se há um token de agente envolvido. Tudo que você precisa é de uma chave API.

⚙️

Personalize Tudo

Controle granular sobre seus agentes de IA: objetivo, descrição, workers, o que o agente vê (estado) e o que pode fazer (ações/funções).

🔌

Integrações Externas

Importe conjuntos de dados externos como contexto adicional para o agente, expandindo suas capacidades de tomada de decisão.

🧩

Plugins Prontos

Plugins prontos para uso com nossos parceiros, incluindo transações on-chain, geração de imagens, e muito mais.

👥

Contribuições da Comunidade

Sistema de plugins aberto - você é bem-vindo para contribuir e expandir as funcionalidades do framework!

🚀

Alto Desempenho

Construído com os melhores modelos de fundação, incluindo Llama 3.1, DeepSeek e Qwen para máxima performance.

Exemplo Rápido

# Instalar o SDK
pip install game_sdk

# Criar um agente simples
from game_sdk.game.agent import Agent
from game_sdk.game.worker import Worker

# Configurar o agente
agent = Agent(
    id="meu-agente",
    description="Um agente inteligente que ajuda com tarefas",
    goal="Auxiliar usuários de forma eficiente",
    workers=[worker_config],
    api_key="sua_chave_api"
)

# Executar o agente
response = agent.run(max_iterations=5)
print(response)
